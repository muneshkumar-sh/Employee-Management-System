# Employee Management System

The **Employee Management System** is a Java Swing + MySQL project that manages employee records with CRUD operations.  

## Features
- Add, view, search, edit, and delete employee records  
- Categorization: Manager, Salesman, Developer  
- User-friendly GUI with confirmation dialogs  

## Technologies
- Java (Swing GUI)  
- MySQL Database  
- JDBC  

## Setup
1. Create a database named `employee_db`.  
2. Create a table `employees` according to your schema.  
3. Update DB credentials in `DatabaseConnection.java`.  
4. Run `EmployeeProject.java` to start the application.
